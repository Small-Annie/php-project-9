# Description

["Page Analyzer"](https://php-project-9-2zw6.onrender.com) — a website that analyzes given pages for SEO suitability.

# Требования \ Requirements

* Mac / Linux
* PHP ^8.4
* Composer
* Make
* PostgreSQL
* Docker (требуется для деплоя)

# Установка и настройка \ Installation and setup

```
git clone git@github.com:Small-Annie/php-project-9.git
cd php-project-9

make install
make start
```
